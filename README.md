# 本地收支 BI 看板

一个只在本机浏览器运行的信用卡 / 借记卡账单分析看板。

## 使用

```bash
npm install
npm run dev
```

打开终端显示的本地地址后，导入 `.xls`、`.xlsx` 或 `.csv` 账单文件即可。

## 支持的账单字段

- 信用卡：交易日期、记账日期、交易摘要、卡号末四位、卡片类型、交易币种、交易金额、原始交易金额&币种。
- 借记卡：账号、账户类型、日期范围、币别号、序号、交易日期、交易类型、存入金额、取出金额、余额、对方全称、交易对手账号、附言。

## 数据口径

- 月度统计按自然月计算。
- 信用卡消费按交易日期归属月份。
- 借记卡存入金额计为收入，取出金额计为支出。
- 信用卡还款、账户互转、理财申购赎回默认作为转账，不进入消费支出。
- 每月 TOP N 按单笔支出金额排序，默认 TOP 10，可在界面调整。

## 隐私与安全

数据保存在浏览器 IndexedDB 中，不上传服务器。Excel 解析使用 `xlsx`，它支持老式 `.xls` 银行账单；npm audit 对该包有已知提示且暂无 npm 可用修复版本。建议只导入来自银行、且你确认可信的本地账单文件。
